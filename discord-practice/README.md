# Discord Practice
## Instruction
1. Add .env file on root folder, and write down your DISCORD_TOKEN in it. Ex. `DISCORD_TOKEN=XXXX`.
2. Start dev server: `npm run start:dev`. It can auto restart server after save file.

## Features
1. Luck
    * Check daily luck
    * Command: !luck, !運勢
2. Play Audio
    * Play youtube song
    * Don't have song queue
    * Command: !p
